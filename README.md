# homebrew-tap

Homebrew formulas for [michaelsieminski](https://github.com/michaelsieminski)'s tools.

## Usage

```bash
brew install michaelsieminski/tap/ninode
```

Or tap once and install by short name:

```bash
brew tap michaelsieminski/tap
brew install ninode
```

## Available formulas

| Formula  | Description                                          |
| -------- | ---------------------------------------------------- |
| `ninode` | Terminal-based server monitoring (React 19 + OpenTUI) |

## Updating

Formulas in this tap are bumped automatically by each upstream project's
release workflow. To bump manually, edit `Formula/<name>.rb` and open a PR.
